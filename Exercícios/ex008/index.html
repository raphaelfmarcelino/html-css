Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b></b> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong></strong>.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i></i> (não semântica).

Nesta frase, temos um termo em ênfase, usando a tag <em></em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark></mark>.

E no outro parágrafo temos, outro texto marcado no final.

Texto grande e pequemo

Estamos criando um texto grande e um texto pequeno nesse parágrafo, usando as tags <small></small> e <big></big>.

Texto Deletado

Podemos chamar um texto como excluído para inicar que ele deve ser lido, mas não considerado, usando a tag <del></del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins></ins>.

Existe também o sublinhado com a tag <u></u> (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup></sup>.

Texto Subscrito

Para Inserir coisas do tipo H2O, usando a tag <sub></sub>.
